RPG.PY Text-RPG

Problem:

I wanted to make a text-rpg.

Solution:

About one and a half year ago I decided I wanted to make a text-rpg that is available to be run through the terminal.

Even though I haven't been working on it constantly, it has been a very good learning experience and I am constantly changing/refactoring code

Currently up to about 3000 lines of code.

The current features can be found on www.niklaslund.me


Technologies used:

  • Python 3.5
    • CMD
      • For the easy commandline interface and errorhandling
    • Colorama
      • To get away from the black and white and highlift different things (speech, foes, items etc)
