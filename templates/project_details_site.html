Internal site for searching and displaying IP-adresses

Problem:

We have a lot of machines, servers and switches, and to find their IP:s we would need to search through different excelfiles depending on which network it belonged to.


Solution:

I wrote an internal site in which the backend searched through excelfiles for the machine and displayed the IP.


Technologies used:

  • Python 3.5
    • Flask
    • Openpyxl
  • HTML
  • CSS
    • Bootstrap 3
  • Javascript
    • JQuery
