Automated Email

Problem:

I had a raspberry pi which I have used to gather chat-data from counterstrike tournaments on twitch which didn't really have a purpose anymore


Solution:

I wrote a script that would send me an email each morning before I went to work with:

  • A random comicstrip from garfieldminusgarfield.net
  • Train-information from SL
  • Weather information from SMHI

Technologies used:

  • Python 3.5
    • Requests
  • SMHI API
  • SL API
